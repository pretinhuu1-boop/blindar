## Compromissos de v1.0 LTS

- **API estável** por 12 meses (breaking changes só em v2.0)
- **Suporte de patches de segurança** por 18 meses
- **Compatibilidade** com Node 20+ e Bun 1.2+ garantida
- **Schemas** versionados com migration automática
- **Deprecation policy** clara (2 minor de aviso)

## Princípios não-negociáveis pra v1.0

1. **Zero falsos positivos críticos** em projetos canonical (Next.js starter, etc.)
2. **Tempo de execução < 30s** pra `check --fast` em qualquer projeto razoável
3. **Determinismo**: mesmo input = mesmo output (sem AI nondeterminism em decisões blocking)
4. **Auto-fix nunca destrutivo**: sempre cria backup, sempre git-diff-friendly
5. **Honesto**: se não consegue verificar algo, diz "skipped" — não inventa "passed"
